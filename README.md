## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- serve-favicon

## Getting Started

#### Clone the respository
```git clone https://github.com/fr3chtoa45t/tinyapp```
#### Install dependencies
```npm install```
#### Run the server
```npm start```
##### ***Note: the default port is set to 8080***
